# Code of Conduct

## Overview

To ensure an open, welcoming, and creative environment for everyone,  
Arden Engine follows the **Contributor Covenant Code of Conduct (v2.1)**.

This applies to all project spaces, including GitHub issues, pull requests,  
discussions, documentation, and any external community channels.

---

## Our Pledge

We pledge to make participation in this project a harassment-free experience  
for everyone, regardless of:

- age  
- body size  
- visible or invisible disability  
- ethnicity  
- gender identity or expression  
- level of experience  
- nationality  
- personal appearance  
- race  
- religion  
- sexual identity and orientation  

We aim to create an environment of respect, collaboration, learning,  
and professional discussion.

---

## Our Standards

### Examples of positive behavior:
- Respectful and constructive communication  
- Being open to different viewpoints and feedback  
- Giving helpful explanations when discussing architecture or code  
- Helping newcomers understand the project  
- Focusing on what's best for the community and engine development

- Personal attacks, insults, trolling, or harassment

## Enforcement Responsibilities

Project maintainers are responsible for:

- clarifying acceptable behavior  
- taking action in response to misconduct  
- removing inappropriate comments, commits, or contributions  
- issuing temporary or permanent bans when necessary

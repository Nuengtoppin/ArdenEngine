# Coordinates & Rotation

Этот модуль описывает, как в двигателе Dev задаются адреса вокселей и контейнеров (сектор → чанк → окточанк → субчанк → воксель), и как над содержимым выполняются **дискретные повороты** без нарушения топологии.

Цели:

- единый язык адресации (`S0+ / A0 / O0 / s6 / v(11|23|2)`);
- независимость адресации от конкретных размеров контейнеров;
- локальные повороты внутри N×N×N без плавающей арифметики;
- сохранение инварианта «1 адрес → 1 воксель».

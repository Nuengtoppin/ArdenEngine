## 1. 2025-12-06 — MVP0-lab: DUN-шар, аквариум и топологическая сетка

- Собран базовый `Mvp0ScenePlugin`, который отвечает за MVP0-lab:
  стартовые системы (камера, свет, аквариум) и update-цикл
  (`debug_grid`, fly-камера, спавн DUN-контейнеров по клавише `E`).
- Оформлен DUN как отдельный компонент  
  `Dun { kind, chunk_coord, voxel }`, а спавн динамического 32³
  воксельного шара вынесен в модуль `dun::spawn`.  
  Шар живёт в локальной решётке `VoxelGrid` и снаружи видится
  как один объект топологии.
- Сделан простой мезер (`voxel::mesher` + `render::mesh_builder`),
  который собирает треугольный меш только по занятым вокселям.
- Реализован модуль `physics::collider_builder` с двумя стратегиями:
  грубый коробочный коллайдер по размеру грида и точный `trimesh` по мешу.
  На текущем MVP0 используется коробочный вариант.
- Настроена сцена-«аквариум»: пол 512×512 и невидимые стены-борта
  (фиксированные коллайдеры), чтобы DUN-шары физически сыпались
  и оставались внутри объёма.
- Добавлена fly-камера (мышь + WASD/Shift) и гибкая debug-сетка:
  уровни Region / Chunk / Octochunk / Minor, пунктирами 32×32,
  LOD по высоте камеры и крестики в узлах Octochunk по `F3`.
- Проведён ручной стресс-тест: можно заспавнить сотни DUN-контейнеров,
  они корректно падают, отскакивают внутри аквариума и визуально
  совпадают с коллайдерами.

# **Arden Engine**
### *Hybrid Voxel-Core Engine — Rust + Bevy — R&D Architecture*

---

## 🌍 Overview (EN)

**Arden Engine** is an experimental hybrid voxel–vector engine designed for  
**R&D, simulation, procedural generation, dynamic worlds, and destruction systems.**

It combines:

- voxel topology (Region → Chunk → Octochunk → Voxel)  
- vector/mesh structures  
- strict addressing and rotation (Topology / Route / Rotation)  
- dynamic containers (DYN)  
- EQ-core with HAOS adaptive optimization  
- a modular BUS → GPU pipeline  

The project is open to ideas, discussions, experimentation, and exploration.
